# App Boilerplate

This is a basic multi-tenant application based on the [T3 Stack](https://create.t3.gg/), bootstrapped with `create-t3-app`.

## What Technology is Used

- [Next.js](https://nextjs.org)
- [NextAuth.js](https://next-auth.js.org)
- [Prisma](https://prisma.io)
- [Tailwind CSS](https://tailwindcss.com)
- [tRPC](https://trpc.io)
